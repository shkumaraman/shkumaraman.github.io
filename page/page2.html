CSS सीखें - Basic to Advanced

CSS (Cascading Style Sheets) वेब पेज के लुक और लेआउट को डिज़ाइन करने के लिए उपयोग किया जाता है। यह HTML के साथ मिलकर काम करता है।

1. CSS Introduction

CSS का उपयोग वेबसाइट के डिज़ाइन और लेआउट को नियंत्रित करने के लिए किया जाता है। CSS को तीन प्रकार से जोड़ा जा सकता है: Inline CSS, Internal CSS और External CSS।

2. CSS Syntax

CSS नियम (Rule) एक Selector और उसके अंदर Properties और Values से बनता है।

h1 {
    color: blue;
    font-size: 24px;
    font-weight: bold;
}

3. CSS Selectors

Selectors का उपयोग HTML एलिमेंट्स को स्टाइल देने के लिए किया जाता है।

p { color: red; }   /* Element Selector */
#id { color: green; } /* ID Selector */
.class { color: blue; } /* Class Selector */
* { margin: 0; } /* Universal Selector */

4. Box Model

CSS में Box Model बहुत महत्वपूर्ण है। यह एक एलिमेंट के content, padding, border और margin को परिभाषित करता है।

div {
    width: 200px;
    height: 100px;
    padding: 10px;
    border: 2px solid black;
    margin: 10px;
    background-color: lightgray;
}

5. Flexbox

Flexbox एक लेआउट सिस्टम है जिससे एलिमेंट्स को आसानी से arrange किया जा सकता है।

.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
    height: 100vh;
}

6. CSS Grid

CSS Grid एक powerful layout system है जिससे complex layouts बनाए जा सकते हैं।

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 10px;
}

7. CSS Animations

CSS में @keyframes का उपयोग कर animations बनाए जा सकते हैं।

@keyframes example {
    from { background-color: red; }
    to { background-color: yellow; }
}

.box {
    animation: example 2s infinite;
}

8. Media Queries (Responsive Design)

Responsive Design के लिए Media Queries का उपयोग किया जाता है।

@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

9. Advanced CSS Concepts

CSS में Pseudo-classes, Pseudo-elements, और Variables जैसे advanced फीचर्स भी होते हैं।

button:hover {
    background-color: blue;
}

निष्कर्ष (Conclusion)

CSS वेबसाइट डिज़ाइन के लिए आवश्यक है। ज्यादा प्रैक्टिस करने से आप CSS को मास्टर कर सकते हैं। Happy Coding!
