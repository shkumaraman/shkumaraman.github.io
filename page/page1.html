Python सीखें - Basic to Advanced

Python एक high-level, interpreted और object-oriented programming भाषा है। इसे पढ़ना और लिखना बहुत आसान है, इसलिए beginners के लिए यह perfect language है।

1. Python Introduction

Python को Guido van Rossum ने 1991 में develop किया था। यह एक general-purpose language है, जिसका उपयोग web development, data science, automation, और AI जैसी fields में होता है।

2. Python Installation

सबसे पहले आपको Python install करना होगा। इसे python.org से डाउनलोड करें और इंस्टॉलेशन गाइड को फॉलो करें।

3. पहला Python प्रोग्राम (First Program)

print("Hello, World!")

4. Variables and Data Types

x = 10       # Integer
y = 3.14     # Float
name = "Amit"  # String
is_python_fun = True  # Boolean
print(name)

5. Conditional Statements

age = 18
if age >= 18:
    print("आप वोट कर सकते हैं।")
else:
    print("आप अभी वोट नहीं कर सकते।")

6. Loops (for & while)

# For loop example
for i in range(5):
    print("Hello", i)

# While loop example
count = 0
while count < 5:
    print("Counting:", count)
    count += 1

7. Functions in Python

def greet(name):
    print("Hello,", name)

greet("Amit")

8. Lists and Tuples

fruits = ["Apple", "Banana", "Mango"]
print(fruits[0])  # Output: Apple

numbers = (1, 2, 3, 4)
print(numbers[2])  # Output: 3

9. Dictionaries in Python

student = {
    "name": "Amit",
    "age": 20,
    "course": "Python"
}
print(student["name"])  # Output: Amit

10. Object-Oriented Programming (OOP)

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print("Hello, my name is", self.name)

p = Person("Amit", 25)
p.greet()

11. File Handling in Python

# Writing to a file
with open("test.txt", "w") as f:
    f.write("Hello, this is a test file.")

# Reading from a file
with open("test.txt", "r") as f:
    print(f.read())

12. Python Modules and Libraries

import math
print(math.sqrt(16))  # Output: 4.0

13. Exception Handling

try:
    x = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")

14. Web Scraping with Python

import requests
from bs4 import BeautifulSoup

url = "https://www.example.com"
response = requests.get(url)
soup = BeautifulSoup(response.text, "html.parser")
print(soup.title.text)

15. Python for Data Science

import pandas as pd
import numpy as np

data = {
    "Name": ["Amit", "Ravi", "Sita"],
    "Age": [20, 22, 24]
}
df = pd.DataFrame(data)
print(df)
