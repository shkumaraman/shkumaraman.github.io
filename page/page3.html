JavaScript सीखें - Basic to Advanced

JavaScript एक high-level, interpreted और object-oriented programming भाषा है। यह वेब डेवलपमेंट के लिए बहुत महत्वपूर्ण है और इसे हर वेब डेवलपर को सीखना चाहिए।

1. JavaScript Introduction

JavaScript को Brendan Eich ने 1995 में develop किया था। यह वेब पेजों को interactive बनाने के लिए उपयोग की जाती है और लगभग हर आधुनिक ब्राउज़र इसे सपोर्ट करता है।

2. JavaScript Installation

JavaScript को इस्तेमाल करने के लिए आपको किसी विशेष इंस्टॉलेशन की आवश्यकता नहीं होती, यह सीधे वेब ब्राउज़र में चलता है। आप Chrome DevTools या किसी भी टेक्स्ट एडिटर (VS Code, Sublime Text) का उपयोग कर सकते हैं।

3. पहला JavaScript प्रोग्राम (First Program)

console.log("Hello, World!");

4. Variables and Data Types

let x = 10;        // Number
let y = 3.14;      // Float
let name = "Amit"; // String
let isJSFun = true; // Boolean
console.log(name);

5. Conditional Statements

let age = 18;
if (age >= 18) {
    console.log("आप वोट कर सकते हैं।");
} else {
    console.log("आप अभी वोट नहीं कर सकते।");
}

6. Loops (for & while)

// For loop example
for (let i = 0; i < 5; i++) {
    console.log("Hello", i);
}

// While loop example
let count = 0;
while (count < 5) {
    console.log("Counting:", count);
    count++;
}

7. Functions in JavaScript

function greet(name) {
    console.log("Hello,", name);
}
greet("Amit");

8. Arrays and Objects

let fruits = ["Apple", "Banana", "Mango"];
console.log(fruits[0]);  // Output: Apple

let student = {
    name: "Amit",
    age: 20,
    course: "JavaScript"
};
console.log(student.name);  // Output: Amit

9. DOM Manipulation

document.getElementById("demo").innerHTML = "Hello JavaScript!";

10. Event Handling

document.getElementById("btn").addEventListener("click", function() {
    alert("Button Clicked!");
});

11. Object-Oriented Programming (OOP)

class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }
    greet() {
        console.log("Hello, my name is", this.name);
    }
}
let p = new Person("Amit", 25);
p.greet();

12. Fetch API (AJAX)

fetch("https://api.example.com/data")
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error("Error:", error));

13. JavaScript ES6+ Features

let arrowFunction = (name) => console.log("Hello,", name);
arrowFunction("Amit");
