Documentation - Code Editor

Documentation

  • 1. Code Editing

    Supports HTML, CSS, JavaScript, and TXT files with syntax highlighting. You can easily search for text and edit the code seamlessly.

  • 2. File Handling

    • Import files via local upload or URL link.
    • Download files individually or as a ZIP file containing multiple files.

  • 3. User Interface & Experience

    Provides Dark Mode and Light Mode toggle, offering a comfortable and customizable UI for the user.

  • 4. Code Management

    • Copy and paste code within the editor easily.
    • Shortcut snippets and comments are available to speed up your workflow.

  • 5. Sharing & Collaboration

    Generate a shareable link for your code and allow others to view or collaborate.

  • 6. Live Preview

    See the live output of your code (HTML, CSS, and JavaScript) directly within the editor, making real-time testing easier.
