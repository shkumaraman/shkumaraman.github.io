How to Use TalkRush

TalkRush is a peer-to-peer chat application that connects directly between browsers without servers.

Getting Started

  1. Wait for your Peer ID to generate (shown above)
  2. Share your ID with the person you want to chat with
  3. Or enter their ID to connect to them
  4. Once connected, start chatting!

Features

  • End-to-end encrypted communication
  • Typing indicators
  • Connection status monitoring
  • Works on mobile and desktop

Tips

  • Both users need to be online at the same time
  • Refresh the page to get a new Peer ID
  • Copy your ID to share it easily
